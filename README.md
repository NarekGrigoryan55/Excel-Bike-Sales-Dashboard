# Bike Sales Dashboard (Excel)

## Project Overview

This project analyzes customer demographics and bike purchasing behavior using Microsoft Excel. An interactive dashboard was created to identify trends and factors influencing bike purchases.

## Dataset

The dataset includes customer information such as:

- Gender
- Marital Status
- Education
- Region
- Income
- Age Bracket
- Commute Distance
- Bike Purchase Status

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Dashboard Features

### Average Income Per Purchase
Compares average income between customers who purchased a bike and those who did not.

### Customer Age Bracket Analysis
Shows bike purchase behavior across different age groups.

### Customer Commute Analysis
Analyzes the relationship between commute distance and bike purchases.

### Interactive Filters
The dashboard includes slicers for:
- Marital Status
- Education
- Region

Users can dynamically filter the dashboard and explore different customer segments.

## Key Insights

- Customers with higher incomes tend to purchase bikes more frequently.
- Middle-aged customers show the highest purchase activity.
- Commute distance appears to influence purchasing decisions.
- Regional and educational differences can be explored through interactive filters.
